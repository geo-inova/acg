ACG.Core.Objects Namespace
Namespace pertaining to AreaCAD-GIS object classes. Each type, besides AcgObject (which is abstract), represents one physical structure in nature. Any concrete class must inherit AcgObject class, or directly implement IAcgObject interface.
Classes
  ClassDescription
AcgBuilding
Represents a permanent residental, commercial or utility structure.
AcgObject
Represents base class for any AreaCAD-GIS object.
AcgParcel
Represents a cadastral parcel, basic building block of national land registry.
